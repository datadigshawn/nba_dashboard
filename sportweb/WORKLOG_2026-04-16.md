## 🔑 關鍵技術決策

### 決策 1：放棄 DOM 解析 → 全面走 API
**原因：**
- 頁面是 React SPA，HTML 只有框架（`<div id="root">` + 一堆 `<script>`）
- DOM 動態生成、class 名用 hash（例如 `sc-ePDLzJ fjsNTT`），版本改就失效
- API JSON 結構穩定、含完整 `idfoevent` / `idfomarket` ID

**結果：** 解析邏輯短 80%、穩定度 +300%

### 決策 2：用 `page.evaluate` 發 fetch 而非 `ctx.request.post`
**原因：**
- `ctx.request.post` 是獨立 HTTP client，**不帶瀏覽器 CF cookies**
- `page.evaluate` 裡的 JS 執行等同瀏覽器環境，CF 放行

**學到：** Cloudflare 會檢查 header + TLS fingerprint + JS context，光有 cookie 不夠

### 決策 3：bootstrap 智能等待
**原因：**
- 固定秒數倒數可能太早（CF 還沒過）或太慢（浪費時間）
- 用 `page.title()` + 頁面內容偵測是否過關

**關鍵判斷條件：**
- title 不含「請稍候」/「Just a moment」
- HTML 含球隊字詞或市場類別字詞（「不讓分」、「讓分」、「大/小」）
